


          
# core_top.v 模块连线总表

## 1. 模块列表概览

| 序号 | 模块名 | 实例化名 | 功能 |
|------|--------|----------|------|
| 1 | fetch | `u_fetch` | 指令取回与分发 |
| 2 | spgemm_decode | `u_spgemm_decode` | SpGEMM指令解码 |
| 3 | axi_read_mux | `u_read_mux` | AXI读通道复用 |
| 4 | global_buffer | `u_global_buffer` | 片上数据缓存 |
| 5 | load | `u_load` | 矩阵数据加载 |
| 6 | scheduler | `u_scheduler` | PE任务调度 |
| 7 | pe_top | `u_pe[0:`N_PE-1]` | 脉动阵列PE单元 |
| 8 | c_csr_writer | `u_csr_writer` | 结果CSR格式化 |
| 9 | output_scratchpad | `u_outbuf` | 输出数据缓存 |
| 10 | store | `u_store` | 结果写回内存 |

---

## 2. 各模块详细连线

### 2.1 fetch (指令取回模块)

| 端口方向 | 端口名 | 位宽 | 连接来源/目标 |
|----------|--------|------|---------------|
| **输入** | `launch` | 1 | ← `cr_launch` (顶层控制) |
| **输入** | `ins_baddr` | `AXI_ADDR_WIDTH` | ← `ins_baddr` (顶层) |
| **输入** | `ins_count` | 16 | ← `ins_count` (顶层) |
| **输出** | `m_axi_arvalid` | 1 | → `fetch_arvalid` → axi_read_mux |
| **输入** | `m_axi_arready` | 1 | ← `fetch_arready` ← axi_read_mux |
| **输出** | `m_axi_araddr` | `AXI_ADDR_WIDTH` | → `fetch_araddr` → axi_read_mux |
| **输出** | `m_axi_arlen` | 8 | → `fetch_arlen` → axi_read_mux |
| **输入** | `m_axi_rvalid` | 1 | ← `fetch_rvalid` ← axi_read_mux |
| **输出** | `m_axi_rready` | 1 | → `fetch_rready` → axi_read_mux |
| **输入** | `m_axi_rdata` | `AXI_DATA_WIDTH` | ← `fetch_rdata` ← axi_read_mux |
| **输入** | `m_axi_rlast` | 1 | ← `fetch_rlast` ← axi_read_mux |
| **输出** | `ld_inst_valid` | 1 | → `fetch_ld_valid` |
| **输入** | `ld_inst_ready` | 1 | ← `fetch_ld_ready` (来自load模块) |
| **输出** | `ld_inst` | `INST_WIDTH` | → `fetch_ld_inst` |
| **输出** | `sp_inst_valid` | 1 | → `fetch_sp_valid` |
| **输入** | `sp_inst_ready` | 1 | ← `fetch_sp_ready` = 1'b1 |
| **输出** | `sp_inst` | `INST_WIDTH` | → `fetch_sp_inst` |
| **输出** | `st_inst_valid` | 1 | → `fetch_st_valid` |
| **输入** | `st_inst_ready` | 1 | ← `fetch_st_ready` (来自store模块) |
| **输出** | `st_inst` | `INST_WIDTH` | → `fetch_st_inst` |
| **输出** | `sch_inst_valid` | 1 | → `fetch_sch_valid` |
| **输入** | `sch_inst_ready` | 1 | ← `fetch_sch_ready` = 1'b0 |
| **时钟** | `aclk` | 1 | ← 顶层 `aclk` |
| **复位** | `aresetn` | 1 | ← 顶层 `aresetn` |

**复用策略**：优先级 Load > Fetch

### 2.6 scheduler (PE任务调度器)

| 端口方向 | 端口名 | 位宽 | 连接来源/目标 |
|----------|--------|------|---------------|
| **输入** | `start` | 1 | ← `sched_start` = `(state == STATE_SCHEDULE)` |
| **输出** | `done` | 1 | → `sched_done` → 状态机 |
| **输入** | `M` | `MAX_DIM_BITS` | ← 内部寄存器 `M` |

| **输入** | `N` | `MAX_DIM_BITS` | ← 内部寄存器 `N` |

| **输入** | `a_col_idx_base` | 16 | ← `a_col_idx_sram` |
| **输入** | `b_row_ptr_base` | 16 | ← `b_row_ptr_sram` |
| **输出** | `pe_row_start` | `N_PE`×`MAX_DIM_BITS` | → 各PE `row_start` |
| **输出** | `pe_row_end` | `N_PE`×`MAX_DIM_BITS` | → 各PE `row_end` |
| **输出** | `pe_a_ptr_start` | `N_PE`×16 | → 各PE `a_ptr_start` |
| **输出** | `pe_a_ptr_end` | `N_PE`×16 | → 各PE `a_ptr_end` |
| **输出** | `pe_task_valid` | `N_PE` | → 各PE `start` 条件之一 |
| **输出** | `gbuf_rd_en` | 1 | → `gbuf_rd_en` → global_buffer |
| **输出** | `gbuf_rd_addr` | `GBUF_DEPTH_LOG` | → `gbuf_rd_addr` → global_buffer |
| **输入** | `gbuf_rd_data` | `BANK_BLOCK_SIZE` | ← `gbuf_rd_data` ← global_buffer |
| **输入** | `gbuf_rd_valid` | 1 | ← `gbuf_rd_valid` ← global_buffer |
| **时钟** | `aclk` | 1 | ← 顶层 `aclk` |
